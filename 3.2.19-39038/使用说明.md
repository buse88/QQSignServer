该版本是编译好的3.2.19-39038，仅支持amd64位

1.把sign-server.tar 和docker-compose.yml 上传到服务器，执行导入docker映像命令 

~~~
curl -L -O https://github.com/buse88/QQSignServer/releases/download/v3.2.19/sign-server.tar
~~~
~~~
cd sign-server.tar路径
docker load -i sign-server.tar
~~~
2.执行docker compose命令创建docker容器
~~~
docker compose up -d
~~~
